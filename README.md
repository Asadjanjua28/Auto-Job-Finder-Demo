## 🛠️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/Asadjanjua28/auto-job-finder-demo.git
cd auto-job-finder-demo
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the App

```bash
streamlit run auto_job_finder_demo.py
```
